# FitTrack

FitTrack is a full-stack fitness web application that helps users track meals, discover recipes, and plan nutrition.

## Tech Stack
- React
- Node.js
- Express
- MongoDB

## Features
- Recipe recommendation system
- Meal planner
- Authentication
- Nutrition summary

## Project Structure
backend/ – API and database logic  
frontend/ – React UI

## Setup
1. Clone the repo
2. Run `npm install`
3. Start backend and frontend
